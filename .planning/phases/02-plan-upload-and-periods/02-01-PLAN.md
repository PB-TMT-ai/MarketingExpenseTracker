SheetJS read pattern (D2-06 + RESEARCH §1):
```
import * as XLSX from "xlsx";
const wb = XLSX.read(buffer, { type: "array", cellDates: true });
const first = wb.Sheets[wb.SheetNames[0]];
const rows = XLSX.utils.sheet_to_json<unknown[]>(first, {
  header: 1,        // array-of-arrays — header row at index 0
  raw: false,       // formatted strings, no numeric serials for dates/currency
  defval: "",       // null/undefined cells → "" so column positions stay aligned
  blankrows: false, // skip blank rows
});
```

SheetJS template-write pattern (RESEARCH §2):
```
import * as XLSX from "xlsx";
const sheet = XLSX.utils.aoa_to_sheet([headers]);
const wb = XLSX.utils.book_new();
XLSX.utils.book_append_sheet(wb, sheet, "Plan");
const out = XLSX.write(wb, { type: "array", bookType: "xlsx" }) as ArrayBuffer;
// callers wrap as new Blob([out], { type: "application/vnd.openxmlformats-officedocument.spreadsheetml.sheet" })
```
